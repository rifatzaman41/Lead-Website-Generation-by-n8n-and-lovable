# Lead-Website-Generation-by-n8n-and-lovable

🚀 AI Lead Generation Web App

An AI-powered lead generation web application that automates the process of finding business leads across different countries, cities, and niches.

🔗 Live App: https://lead-flow-magic-32.lovable.app

📌 Overview

This project is built to solve a real-world problem: manual lead generation is slow and repetitive.

The application allows users to:

Generate unlimited lead queries

Target specific countries, cities, and industries

Automate data collection using workflows

Store and manage generated leads

⚙️ Tech Stack

Frontend

Lovable (No-code web app builder)

Backend / Automation

n8n (Workflow automation)

Integrations

Apify (Data scraping & lead extraction)

Google Sheets (Database & storage)

Email (Lead delivery / notifications)

Webhooks (Trigger-based automation)

🔥 Features

🔐 User Authentication (Login system)

🌍 Location-based lead generation (Country / City / Niche)

🤖 AI-powered query generation

🔄 Automated workflows using n8n

📊 Data storage with Google Sheets

📩 Email integration for notifications or lead delivery

⚡ Real-time processing via Webhooks
